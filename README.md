# Тестовое задание в Scala
## Задание
Дан список, содержащий элементы трёх типов:
- целые
- строки
- списки целых

Необходимо вывести на экран все значения меньше 38 в виде одной строки
## Пример
```python
 a = (1, 2, 3, 45, "7", [12, 45, 1], 23, "56")
```
Для примера вывод должен быть таким:
```python
1, 2, 3, 7, 12, 1, 23
```
